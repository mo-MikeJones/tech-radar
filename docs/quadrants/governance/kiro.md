## Summary
Kiro is an AI-powered IDE from Amazon designed for cloud-native and serverless development workflows, with deep integration into AWS services.

## Why we're interested
As a heavy AWS user, tooling that understands AWS context and patterns could significantly improve developer productivity on cloud infrastructure and serverless code.

## Current status
Under initial evaluation by a small number of engineers, particularly those working on AWS-native services.

## Evidence & feedback
Strong AWS service awareness. Infrastructure-as-code and serverless use cases show the most differentiation. Still early in evaluation — insufficient data for a broad recommendation.

## Known risks / concerns
Vendor lock-in to AWS ecosystem. Maturity of tooling compared to established alternatives is uncertain. Requires careful assessment before broader rollout.

## Recommended audience
Engineers working primarily on AWS-native services, serverless and infrastructure-as-code.

## Next steps
Complete structured evaluation focused on AWS use cases. Share findings with the broader engineering community. Make a go/no-go recommendation within one quarter.

Cloud Platform Team
