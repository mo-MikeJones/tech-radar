## Summary
Claude Code is Anthropic's AI coding agent that operates in the terminal, capable of reading and editing files, running tests and working autonomously on multi-step tasks.

## Why we're interested
Claude Code represents a differentiated approach to agentic development with strong performance on complex multi-step tasks and excellent code reasoning.

## Current status
Being evaluated by a small number of engineers. Promising early results, particularly on large refactoring and complex debugging tasks.

## Evidence & feedback
Competitive with Copilot on code generation. Strong at reasoning through complex codebases. Early users report it handles multi-step tasks with fewer errors than comparable alternatives.

## Known risks / concerns
Third-party AI service with its own data handling terms. High token usage can be costly at scale. Agentic capabilities require careful oversight and review of all file changes.

## Recommended audience
Senior engineers comfortable with agentic AI tools and willing to actively evaluate and report findings.

## Next steps
Structure a formal evaluation against defined use cases. Compare performance and cost against Copilot Agent Mode. Present findings to the AI tooling working group.
