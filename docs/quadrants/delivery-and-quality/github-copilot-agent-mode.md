## Summary
An agentic mode in GitHub Copilot that allows the AI to autonomously plan, write and execute multi-step coding tasks across files and terminals.

## Why we're interested
Agent mode extends AI assistance beyond single-file completions, enabling end-to-end task automation for scaffolding, refactoring and test generation.

## Current status
Available and recommended. Teams actively using agent mode for multi-file tasks. Guidance on effective use is published on the developer portal.

## Evidence & feedback
Early adopters report significant time savings on repetitive multi-file tasks. Most effective when tasks are clearly scoped and instructions are precise.

## Known risks / concerns
Agents can make unintended changes across the codebase. Human review of all agent-generated changes is essential before committing.

## Recommended audience
Experienced engineers comfortable reviewing AI-generated code changes across multiple files.

## Next steps
Develop internal guides on scoping agent tasks effectively. Gather case studies from teams actively using agent mode.
