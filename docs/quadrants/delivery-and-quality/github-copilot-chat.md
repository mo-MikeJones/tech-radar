## Summary
AI-powered chat interface embedded in the IDE enabling developers to write, explain, refactor and debug code through natural language conversation.

## Why we're interested
Reduces context-switching by bringing AI assistance into the developer's existing workflow, with strong evidence of productivity gains across the industry and early internal adopters.

## Current status
Widely adopted. Available to all licensed engineers. Recommended as the primary AI coding assistant for day-to-day development.

## Evidence & feedback
Teams report 20–30% reduction in time spent on boilerplate code and debugging. High satisfaction scores in quarterly developer experience surveys.

## Known risks / concerns
Risk of accepting AI suggestions without adequate review. Some proprietary code context is sent to external services — review data handling policies before use on sensitive repositories.

## Recommended audience
All software engineers and technical leads.

## Next steps
Publish internal best practice guide. Establish quarterly feedback loops to track adoption quality and code review standards.

Developer Experience Team
