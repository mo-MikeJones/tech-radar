## Summary
Using AI tooling to assist in reviewing pull requests, including automated summary generation, risk assessment and actionable feedback.

## Why we're interested
PR review is a significant time investment. AI assistance can surface risks faster and reduce cognitive load on reviewers, particularly for large or complex changes.

## Current status
In active use. GitHub Copilot Code Review is the adopted standard for AI-assisted PR reviews across engineering teams.

## Evidence & feedback
Teams using AI-assisted review report faster PR cycle times and higher confidence when merging. Particularly effective for large or complex PRs.

## Known risks / concerns
Over-reliance on AI reviews may reduce engineers' own code quality instincts. Must remain a tool to augment, not replace, human review.

## Recommended audience
All engineers involved in code review. Particularly valuable for senior engineers reviewing large volumes of PRs.

## Next steps
Document team guidelines on integrating AI review feedback into the PR workflow. Track impact on PR cycle time and defect rates.
