## Summary
GitHub Copilot is an AI pair programmer that suggests code completions, entire functions and documentation directly inside the developer's IDE.

## Why we're interested
Copilot has become the industry-standard AI coding assistant. Strong adoption evidence and productivity data support broad rollout across all engineering teams.

## Current status
Standard tooling for all engineers. Licences available across the organisation. Integrated into VS Code and JetBrains IDEs.

## Evidence & feedback
Consistent reports of 20–40% productivity improvement on code-writing tasks. Highest impact on boilerplate, tests and documentation.

## Known risks / concerns
Developers should review all suggestions carefully. Copilot does not guarantee correctness, security or adherence to internal coding standards.

## Recommended audience
All software engineers.

## Next steps
Ensure all engineers have access and have completed basic onboarding. Develop advanced usage guides for complex completions and agent mode.

Developer Experience Team
