## Summary
Amazon Bedrock Agents is a managed service for building, deploying and operating AI agents on AWS, with built-in tool use, memory and orchestration capabilities.

## Why we're interested
Bedrock Agents provides a managed, AWS-native platform for running AI agents alongside our existing cloud infrastructure, reducing operational overhead.

## Current status
Under active evaluation. Proof of concept built for an internal use case. Assessing fit, capabilities and cost model.

## Evidence & feedback
Tight AWS integration simplifies deployment. Agent capabilities are functional. Cost model needs careful management at scale. IAM integration for security works well.

## Known risks / concerns
Vendor lock-in to AWS for agent orchestration. Cost can escalate quickly at scale. Less flexibility than self-hosted alternatives for complex orchestration needs.

## Recommended audience
Cloud engineers and AI teams building agent-based automation on AWS infrastructure.

## Next steps
Complete POC evaluation. Compare cost and capabilities against alternative agent platforms. Make a strategic fit decision within one quarter.
